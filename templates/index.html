📖 About Project

AI Enhanced Intrusion Detection System is a Machine Learning based Cyber Security solution developed using Random Forest, Flask, Scikit-learn and the NSL-KDD dataset. The system analyzes network traffic patterns and identifies suspicious activities in real time, helping organizations detect cyber threats before they cause damage.
